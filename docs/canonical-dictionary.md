## 3. Safety Taxonomy (안전성 표준명)

### safety_type 분류
| 코드 | 한글 | 설명 |
|------|------|------|
| `adverse_effect` | 이상반응/부작용 | 복용 시 발생 가능한 부작용 |
| `contraindication` | 금기 | 복용하면 안 되는 상황 |
| `precaution` | 주의사항 | 복용 시 주의가 필요한 상황 |
| `drug_interaction` | 약물 상호작용 | 특정 약물과의 상호작용 |
| `food_interaction` | 식품 상호작용 | 특정 식품과의 상호작용 |
| `overdose` | 과다복용 | 과다복용 시 위험 |
| `population_warning` | 특수 집단 주의 | 임산부/수유부/소아/고령자 주의 |

### severity 분류
| 코드 | 한글 | 설명 |
|------|------|------|
| `critical` | 위험 | 생명 위협, 즉시 중단 필요 |
| `serious` | 심각 | 의료 개입 필요 가능 |
| `moderate` | 중등도 | 불편감, 모니터링 필요 |
| `mild` | 경미 | 일시적, 자연 소실 |

### evidence_level (부작용 3계층)
| 코드 | 계층 | 출처 |
|------|------|------|
| `label` | 1층 확정적 | 규제기관 라벨 경고 |

| `rct` | 2층 문헌 | RCT/체계적 문헌고찰 |
| `observational` | 2층 문헌 | 관찰연구/코호트 |
| `case_report` | 2층 문헌 | 증례 보고 |
| `spontaneous_report` | 3층 신호 | 자발보고 DB (빈도 아닌 신호) |

## 4. Unit Normalization Rules (단위 표준화)

### 기본 단위
| 원료 카테고리 | 기본 저장 단위 | 허용 입력 단위 | 환산 규칙 |
|--------------|---------------|---------------|-----------|
| 비타민 (지용성) | mcg | IU, mcg, mg | 비타민 D: 1mcg = 40IU |
| 비타민 (수용성) | mg 또는 mcg | mg, mcg, g | 1mg = 1000mcg |
| 미네랄 | mg | mg, mcg, g | 원소량(elemental) 기준 저장 |
| 지방산 | mg | mg, g | EPA+DHA 합계량 별도 표기 |
| 프로바이오틱스 | CFU | CFU, 억 CFU | 1억 CFU = 10^8 CFU |
| 허브 | mg | mg, g | 추출물 기준, 활성 성분 함량 병기 |
| 아미노산 | mg 또는 g | mg, g | - |
| 기타 | mg | mg, mcg, g | - |

### 특수 환산 규칙
| 원료 | 규칙 |
|------|------|
| 비타민 D | 1 mcg = 40 IU (D3, D2 동일) |
| 비타민 A | 1 mcg RAE = 3.33 IU (retinol) = 10 IU (beta-carotene) |
| 엽산 | mcg DFE = mcg food folate × 1.0 = mcg folic acid × 1.7 |
| 비타민 E | 1 mg = 1.49 IU (d-alpha) = 2.22 IU (dl-alpha) |
| 칼슘 | 원소 칼슘 기준 (탄산칼슘 40%, 구연산칼슘 21%) |
| 마그네슘 | 원소 마그네슘 기준 (산화Mg 60%, 구연산Mg 16%) |
| 철 | 원소 철 기준 (황산제일철 20%, 푸마르산철 33%) |

### 저장 원칙
1. **원소량(elemental) 기준으로 저장** — 미네랄은 염형과 무관하게 원소량으로 통일
2. **원본 라벨 값 보존** — `product_ingredients.raw_label_name`과 `amount_per_serving`에 원본 그대로 저장
3. **환산 값 별도 필드** — 표준화된 값은 별도 컬럼에 저장 (원본 훼손 방지)
4. **단위 불명확 시 검수 전송** — 단위를 확정할 수 없으면 L1 검수 실패 → QA 수동 확인

## 5. 금지 표현 사전 (초안)

| 금지 표현 | 이유 |

| 의약품 대체 | 의약품 오인 유도 |
| 100% 효과 / 확실한 효과 | 과대광고 |
| 부작용 없음 / 완전 안전 | 허위 표현 |
| 즉시 효과 / 즉각적 | 과대광고 |
